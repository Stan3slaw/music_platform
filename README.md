## Description

Used technologies: [NestJs](https://nestjs.com/), [ReactJs](https://reactjs.org/)

## Setting up MONGODB_URL

```bash
Go to -> backend/
Create -> .env
And paste -> MONGODB_URL = Your MongoDB url
```

## Installation and running backend

```bash
$ cd backend/
$ npm install
$ npm run start
```

## Installation and running client part

```bash
$ cd client/
$ npm install
$ npm run start
```
